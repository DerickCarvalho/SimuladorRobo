Para simular a movimentação do robô, fiz uma espécie de "jogo".




Instruções



1 - Utilize os direcionais cima e baixo para mover o robô para frente e para trás.


2 - Utilize as setas laterais para girar o robô 90 graus para os lados desejados.


3 - O botão "OK" finaliza a simulação, mostrando o resultado do algoritmo e levando o robô de volta à posição inicial.


4 - Os espaços de texto centrais mostram todos os passos que o robô está executando, como também sua orientação e coordenadas finais.
